Readme (v0.1)

Установка библиотеки:

- Подключите tabSignal.js и spa.js на нужные страницы, либо в layout файл,
чтобы использовать библиотеку на всех страницах сразу.

Использование:

- Поместите контент, который изменяется при переходе между страницами в блок с id "page".

- Чтобы добавить AJAX переход на страницу, добавьте ссылке с её адресом класс "spa-link".
Также вы можете добавить класс "spa-link" любому блоку (например, 'navbar') и AJAX переход будет
осуществляться по всем ссылкам, содержащимся в нем.
